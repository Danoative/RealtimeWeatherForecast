MANUALLY AND ONLY RUN THE DASHBOARD:
Requires:
+ Node.js
+ React.js
+ Express.js
+ MongoDB (With MongoDB shell and MongoDb Compass or MongoDB Atlas)

How to run:
1/ Start up the server:
+ Open command prompt (Run as administrator)
+ Navigate to the server folder
+ Enter the command: cd <YOUR PATH>/Dashboard/server
+ Enter command: node index.js to start the server

2/ Run the dashboard
+ Simply run index.html inside client folder
